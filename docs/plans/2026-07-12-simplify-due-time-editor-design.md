# 简化截止时间编辑设计

## 目标

让常见的“今天稍后做”只需展开截止时间、滚轮调整时间并保存；只有跨日期时才操作日期，避免进入独立模态日历和二次确认。

## 方案

- 保留“设置截止时间”开关，未启用时不增加表单噪声。
- 时间使用 `QTimeEdit`，格式保持 `HH:mm`，继续支持滚轮、上下键和分段编辑。
- 日期使用次级 `QDateEdit`，启用 `calendarPopup`；选择日期直接应用，不创建独立 `QDialog`。
- 日期与时间分别显示，避免组合控件中滚轮误改日期。
- 保存时继续组合本地日期和时间并转换为 UTC；编辑、提醒和延后语义不变。

## 边界

- 不增加快捷日期、自然语言解析或新的提醒选项。
- 删除被替代的自定义日历弹窗、图标常量和资源，保持单一路径。
- 保持深浅主题可读性与现有输入校验。

## 验证

- 自动化测试证明控件类型、日历下拉、日期时间组合和关闭截止时间行为。
- GUI 验证新增、编辑、日期下拉、滚轮调时、取消、提醒设置和深色主题。
