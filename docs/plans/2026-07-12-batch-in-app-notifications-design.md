# 多任务到期的软件内汇总提醒：设计

## 目标

多个任务同时或相继到期时，只保留一个软件内提醒窗口。任务在窗口内按列表汇总，用户可以选择后批量完成或推迟，避免模态窗口层层叠加和逐个跳出。

## 已验证根因

当前每个任务都调用 `NotificationDialog.exec()`。模态对话框的嵌套事件循环不会停止每秒定时器，不同任务又只按各自 ID 去重，因此第二个任务会在第一个窗口尚未关闭时继续创建新窗口。离屏复现中，两个任务同时到期时活动提醒 ID 为 `[1, 2]`。

## 约束与非目标

- 只使用软件内窗口与软件自身提醒音；不创建 Windows 系统通知、Toast 或任务到期托盘气泡。
- 不修改任务存储结构、提醒偏移、延后时间计算或主题检测。
- 不把提醒改造成常驻通知中心，不增加历史记录或新的提醒类型。
- 系统托盘原有的最小化说明不属于任务到期提醒，本次不调整。

## 方案比较

### 方案 A：单一非模态汇总窗口（采用）

主窗口收集同一轮检查产生的提醒请求，一次性加入一个非模态 `NotificationDialog`。窗口打开期间的新请求按任务 ID 追加或更新，不创建新窗口。

优点是彻底消除嵌套 `exec()`，主窗口仍可操作；同类任务可一次批量处理，混合任务也能通过选择分步处理。代价是对话框需要管理行状态和向主窗口发送批量操作信号。

### 方案 B：全局队列但仍逐个显示

限制任意时刻只有一个模态窗口，关闭后显示下一个。

实现风险较低，但用户仍需逐个点击，未解决核心体验问题。

### 方案 C：只提示数量并返回主列表

显示“3 个任务到期”的简单软件提示，所有处理都回到主列表。

窗口最简单，但丢失当前快速完成和推迟能力，属于功能倒退。

## 对话框设计

`NotificationDialog` 改为接受和维护多个任务：

- 标题显示“1 个任务需要处理”或“3 个任务需要处理”。
- 每行包含默认选中的复选框、任务文本、截止时间和“提前提醒/已到期”状态。
- 同一任务再次进入时按 ID 更新；“已到期”优先于“提前提醒”。
- 底部提供“完成选中”“选中推迟 15 分钟”与现有延后菜单、“忽略全部”。
- 完成或推迟后仅移除已处理行；未选择任务继续保留。没有剩余任务时窗口自动关闭。
- “忽略全部”关闭软件窗口，不改变任务完成状态；提醒标记已在入队时写入，因此不会立即重弹。

对话框通过信号发送任务 ID 列表与延后时长，不直接写存储。

## 主窗口协调

`tick_update` 直接扫描完整的 `self.todos`，不依赖当前筛选后可见的卡片；可见卡片循环只负责刷新倒计时。一次循环中先收集所有提醒请求并更新去重字段，再统一保存并交给 `_show_notification_batch`：

1. 同轮只唤醒/前置应用一次。
2. 同轮只播放一次软件提醒音；批次含到期任务时使用到期音，否则使用提前提醒音。
3. 已有汇总窗口时调用 `add_or_update_tasks`；没有时创建一个非模态窗口并 `show()`，不再调用 `exec()`。
4. 窗口打开期间新一轮到期任务追加到同一窗口，可按新一轮播放一次软件提醒音，但不新开窗口。
5. 批量完成或推迟在主窗口一次更新所有选中任务、一次保存、一次刷新列表。
6. 编辑、删除或从列表完成任务时，同步移除汇总窗口中的对应行。
7. 主窗口关闭到托盘时保留当前提醒批次；从托盘恢复主窗口时重新显示同一个汇总窗口，只有真正退出应用才关闭并释放提醒窗口。

## 状态不变量

- 任意时刻最多一个任务提醒窗口。
- 提醒请求加入汇总窗口前先写入 `notifiedForReminder` / `notifiedForDue`，防止定时器重入重复入队。
- 同一任务在窗口中最多一行；到期状态覆盖提前提醒状态。
- 批量操作只影响显式选中的任务。
- 任务到期路径不调用 `QSystemTrayIcon.showMessage` 或任何 Windows 通知 API。
- 关闭或忽略汇总窗口后删除 Qt 对象并自动断开主题信号，不在主窗口下累积隐藏子窗口。
- 最小化或关闭到托盘不等同于“忽略全部”，提醒批次必须保留到用户处理、明确忽略或真正退出。

## 验证

- 自动化复现三个同时到期任务只创建一个窗口并包含三行。
- 自动化验证窗口打开期间新增任务追加、ID 去重与状态升级。
- 自动化验证批量完成、批量推迟和忽略不会误改未选择任务。
- 自动化验证软件音每批一次且任务到期路径不调用系统通知。
- 自动化验证任务被筛选隐藏时仍会提醒，反复关闭窗口后不残留 Qt 子对象。
- 实际 GUI 验证同时到期、选择与混合处理、追加任务、关闭恢复、深浅主题和窗口前置。
- 运行完整测试、`python -m compileall todo_app` 与 `git diff --check`。
